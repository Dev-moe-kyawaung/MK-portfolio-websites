# Moe Kyawaung - Portfolio Website

A professional portfolio website showcasing projects, certificates, and skills.

## Features

- Responsive design
- Mobile-friendly
- Certificate showcase
- GitHub repositories integration
- Contact form
- WordPress sites collection
- App links

## Setup

1. Clone the repository
2. Open `index.html` in browser
3. No dependencies required

## Deployment

Push to GitHub Pages:

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/Dev-moe-kyawaung/portfolio.git
git push -u origin main
